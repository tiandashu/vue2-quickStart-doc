    # 安装vue-devtools

    git clone https://github.com/vuejs/vue-devtools.git

npm install

npm run build

// 如果看不见“加载已解压的扩展程序...”按钮，则需要勾选“开发者模式”
游览器输入地址“chrome://extensions/” 进入扩展程序页面，点击“加载已解压的扩展程序...”按钮,
选择vue-devtools>shells下的chrome文件夹。
